Punk Provenance Explorer is a visualization tool that tracks and displays the complete ownership history of each CryptoPunk, showing how they've moved between wallets over time.

## Features

- **Full History** - See every transfer and sale for any punk
- **Visual Timeline** - Ownership changes displayed chronologically
- **Wallet Connections** - Understand the journey each punk has taken
- **Transaction Details** - Links to on-chain data for verification

Part of the Junk Drawer suite of CryptoPunks analytics tools by Pat Doyle, alongside PunkVerse.
